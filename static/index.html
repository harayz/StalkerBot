stalkerbot

View Hacker News user's recent comments and state of mind.

Stalking... This will take a minute.


What is going on here?

1. Use HN API to scrape specified HN user's comments,
2. Use Diffbot to extract subject and tags from linked articles,
3. Use EffectCheck to analyze sentiments from user's comments,
4. Display an amusing view of the user's comments and sentiments.


Complements, suggestions, wise cracks? Discuss on Hacker News. GitHub repository here.
